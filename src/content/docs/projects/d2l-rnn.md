# D2L 序列模型与 RNN 实验探索

## 项目背景

这个项目基于《动手学深度学习》序列模型章节，主要学习时间序列预测、文本预处理、语言模型、RNN 实现和 BPTT。

## 我做了什么

- 复现时间序列预测实验
- 对比单步预测和多步预测
- 学习文本预处理流程
- 构造语言模型训练数据
- 从零实现 RNN
- 使用简洁 API 实现 RNN
- 调整学习率和隐藏单元数量
- 分析困惑度变化
- 总结梯度爆炸和梯度裁剪问题

## 实验探索

我在原教材基础上做了参数调整实验，重点观察：

1. 学习率变化对训练稳定性的影响
2. `num_hiddens` 对模型表达能力的影响
3. 训练轮数对困惑度下降的影响
4. 多步预测中误差累积的问题

## 收获

这次实验让我意识到，深度学习实验不能只看代码能不能跑通，更要关注参数、数据、训练过程和结果分析。
